# A Data-Driven Approach for Optical Reference-Guided Optimization of Millimeter-Wave Imaging

This repository contains the official implementation of **“A Data-Driven Approach for Optical Reference-Guided Optimization of Millimeter-Wave Imaging.”**

---
### Dataset

This repository uses millimeter-wave datasets from [1], [2], and our own collected dataset [3].

[1] S. Wei et al., “3DRIED: A High-Resolution 3-D Millimeter-Wave Radar Dataset Dedicated to Imaging and Evaluation,” *Remote Sensing*, vol. 13, no. 17, p. 3366, 2021.

[2] M. E. Yanik and M. Torlak, “Millimeter-Wave Near-Field Imaging With Two-Dimensional SAR Data,” in *Proc. SRC TechCon*, 2018.

[3] L. Dorje et al., “MilliSARImageNet: A 2D High-Resolution Millimeter-Wave SAR Image Dataset,” in *2026 IEEE 23rd Consumer Communications & Networking Conference (CCNC)*, 2026.
